-- 주요 기능 --

 * 이미지를 캐시해두어 빨리 볼 수 있게 함. 50메가 캐시 메모리 제한.
 * 파일 삭제 기능
 * 실수로 Home, Last 등을 눌러 처음, 마지막 파일로 갔을 때 Backspace 키를 이용한 이전 위치로 돌아가기
 * PageDown(다음 폴더), PageUp(이전 폴더) 를 통한 폴더간의 이동
 * 이미지 회전 기능. 시계방향(키 - ] ), 반시계방향(키 - [ )
 * 언어팩을 이용한 국제화 - 현재 한글, 영어. 자동 인식.
 * 일본 서적을 스캔한 경우. 이미지를 넘길 때 오른쪽 위부터 보여주는 기능

-- 추가할 기능 --

 * 이미지 캐시 용량 옵션을 통해 변경 기능
 * 2장의 이미지를 한번에 보여주는 기능
 * 화면에 맞게 확대 기능
 * Enter 키를 통한 이미지 파일 리스트 기능
 * 각종 OS 별로 테스트. 영문, 한글 포함. 옵션 다이얼로그에 넣기
