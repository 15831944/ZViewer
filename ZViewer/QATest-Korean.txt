
= 배포하기 전에 테스트해봐야할 항목들 =

 * 바이러스 검사
 * GDI handle 누수가 발생하지 않는가 
 * 메모리 누수가 발생하지 않는가
 * 배포파일의 설치가 순조롭게 되는가. 2000, xp, 2003
 * 배포파일의 ReadMe 파일은 정확한 내용인가
 * ZViewerAgent 의 모든 메뉴가 제대로 동작하는가
 * 바탕화면으로 지정이 잘 되는가
 * 그림 파일 많은 곳에서 휠을 빠르게 돌려도 괜찮은가
 * 화면에 맞게 축소가 제대로 작동하는가
 * 화면에 맞게 확대가 제대로 작동하는가
 * 파일 삭제가 정확히 되는가
 * 파일 이동이 정확히 되는가
  * 다른 드라이브로의 파일 이동이 정확히 되는가
 * 단축키로 화면을 훑는 기능(A,S,Z,X) 이 작동하는가.
 * 저장된 옵션이 제대로 복구되고 메뉴 설정도 제대로 되는가
 * 파일 캐시가 제대로 되는가. F1 눌렀을 때 갯수가 제대로 나오는가
 * 화면보다 큰 이미지 파일을 열었을 때 드래그가 부드럽게 잘 되는가.
  * 이 때 마우스 커서가 제대로 손모양으로 나오는가
 * transparent png 파일을 열어서 확대, 축소, 회전해서 체크 배경이 나오나 체크
 * 다른 이름으로 저장 & 확장자 바꾸기가 잘되는가
